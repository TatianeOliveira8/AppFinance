FINANCEAPP - GUIA RÁPIDO

1. DESCOMPACTE:
   Extraia o arquivo Salva_Amiga.zip em uma pasta

2. INSTALE:
   Abra terminal na pasta extraída (FinanceApp)
   
   Windows:
     install_all.bat
   
   Mac/Linux:
     bash install_all.sh
   
   Aguarde 2-5 minutos

3. CONFIGURE:
   Arquivo: backend/.env (linha 3)
     DB_PASSWORD=1234
     ↓ Altere para
     DB_PASSWORD=sua_senha_postgresql
   
   Arquivo: frontend/src/config.ts (linha 7)
     const API_BASE_URL = 'http://192.168.15.12:8000/api';
     ↓ Altere para
     const API_BASE_URL = 'http://SEU_IP_LOCAL:8000/api';
   
   Como descobrir seu IP?
     Windows: ipconfig | procure IPv4 Address
     Mac: ifconfig | procure inet
     Linux: hostname -I

4. RODE BACKEND (Terminal 1):
   cd backend
   source venv/bin/activate          (Mac/Linux)
   venv\Scripts\activate.bat         (Windows)
   python main.py
   
   Esperado: "INFO: Uvicorn running on http://0.0.0.0:8000"

5. RODE FRONTEND (Terminal 2):
   cd frontend
   npm start
   
   Esperado: Aparecerá um QR code

6. TESTE NO CELULAR:
   Instale Expo Go (App Store / Google Play)
   Escaneie o QR code que apareceu no Terminal 2
   Aguarde carregar
